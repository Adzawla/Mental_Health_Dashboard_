# Mental_Health_Dashboard

1- This project analyzes how demographic, work-related, and organizational factors influence employee mental health outcomes. The goal is to uncover insights that guide workplace well-being initiatives and reduce mental health risks at work.

2- Tools & Methods
  -Python (Pandas, NumPy) for data cleaning and statistical correlation analysis
  -Plotly Dash for interactive dashboard development and visualization
  -Exploratory Data Analysis (EDA) to identify relationships between socio-demographic, environmental, and perception-based variables.

3- Results
  -Revealed key factors influencing workplace mental health, including workload, management support, and policy awareness, personal perception
  -Identified conditions that encourage treatment uptake and improve employee well-being.
